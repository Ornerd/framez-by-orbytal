# Framez-by-orbytal

This is an [Expo](https://expo.dev) project with supabase as backend.

## Features

- Supabase Authentication
- Upload images and videos
- Like and unlike posts
- Comment system
- User profiles with avatars
- Home feed displaying posts

Built entirely with React Native and Expo

## Getting started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
